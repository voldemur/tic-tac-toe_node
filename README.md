## Игра «Крестики-нолики» на Node.js

Простая игра крестики-нолики, где первый ход определяется по правилу загадывания чисел от 1 до 100.

### Как запустить

1. Установите Node.js (версии 16+ будет достаточно).
2. Откройте терминал в папке проекта.
3. Запустите сервер:

```bash
npm start
```

4. Откройте в браузере `http://localhost:3000`.

### Правила выбора первого хода

- Игрок за **X (крестики)** и игрок за **O (нолики)** загадывают числа от 1 до 100.
- Игра случайно генерирует число от 1 до 100.
- Тот, чьё число оказалось ближе к случайному, ходит первым.
- Если расстояние одинаковое — считается ничья, нужно ввести новые числа и попробовать ещё раз.

### Управление

- После определения первого хода кликайте мышкой по свободным ячейкам на поле 3×3.
- Фигуры **X** и **O** появляются с плавной анимацией.
- Победная линия подсвечивается, после чего можно начать новую партию кнопкой «Начать заново».
